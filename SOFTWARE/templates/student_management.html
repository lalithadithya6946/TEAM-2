How to use Excel Dataset Management
  • Download the Excel file from your system
  • Edit student data directly in Excel (names, departments, room numbers, etc.)
  • Delete rows to remove students from the system
  • Add new rows to add new students
  • Upload the updated Excel file to sync with the system
  • Students not in the Excel file will be automatically removed
